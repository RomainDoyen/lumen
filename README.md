# Lumen

**Lumen** est une application de **compression d’images** (WebP), de **packs WordPress** (plusieurs images → ZIP, `metadata.json`, Gutenberg) et de **générateur de packs d’icônes**. Le traitement image reste **local** ; la suggestion de métadonnées via **Mistral** est optionnelle (votre clé API).

---

## Documentation

**Tout le détail du projet** — présentation, installation, utilisation, architecture, compilation et distribution — se trouve dans le dossier **`docs/`**.

- Ouvrez [`docs/index.html`](docs/index.html) dans un navigateur (redirection vers le guide complet), ou directement [`docs/build-guide.html`](docs/build-guide.html).

Pour lancer l’app en développement une fois les dépendances installées : `npm start` (voir le guide pour le reste).
